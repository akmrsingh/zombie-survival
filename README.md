# Zombie Survival: Class Defense

A wave-based zombie survival game with 4 unique classes, realistic weapons, and multiplayer support!

## Play Online
[Play on itch.io](https://YOUR_USERNAME.itch.io/zombie-survival)

## Features

### 4 Player Classes
- **Builder** - Builds walls to protect the team (HP: 120)
- **Ranger** - Ranged combat expert with multiple guns (HP: 100)
- **Healer** - Creates healing zones for teammates (HP: 90)
- **Tank** - Heavy firepower with explosive weapons (HP: 180)

### Weapons
- Pistol, Assault Rifle, Sniper Rifle
- Pump Shotgun, SMG
- Rocket Launcher, Grenade Launcher, Minigun

### Wave System
- Progressive difficulty with 4 zombie types
- Normal, Runner, Tank, and Spitter zombies
- Survive as long as you can!

### Bunker Base
- Central safe zone
- Switch classes between waves
- Regroup with teammates

### Multiplayer
- Local co-op (1-4 players)
- Online multiplayer (host/join)

## Controls

| Action | Key |
|--------|-----|
| Move | WASD |
| Aim | Mouse |
| Shoot | Left Click |
| Reload | R |
| Class Ability | Z |
| Change Class | B (in bunker) |
| Switch Weapons | 1-4 or Scroll |
| Pause | ESC |

## Class Abilities

- **Builder**: Build defensive walls (max 10)
- **Ranger**: Rapid fire mode
- **Healer**: Deploy healing zone
- **Tank**: Ground slam (AoE damage)

## Installation

### Requirements
- Python 3.8+
- Pygame 2.0+

### Run Locally
```bash
pip install pygame
python3 game.py
```
